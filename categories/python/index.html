爬虫初探

爬虫初探

昨天我萌生出了写爬虫的想法，再加之我以前学过一些前端的知识，所以我今天上午开始了对爬虫的学习。

点子

随便找了个网站：豆瓣读书网

目标

输入ISBN号返回书的信息

实现

  1. 寻找规律
    我发现，豆瓣的输入都是通过网址后边的数据传输的，所以我只需要将输入的ISBN号连接在网址中就行了。
    但是返回的网页内容是类似于百度的索引排列。所以我得选中第一个来进行爬取。但是当我将html保存后，却发现网页时动态加载的，很多的资源需要再次发送请求才能得到（ajax）。
    要解决这个问题，我选择了模拟浏览器来进行操作，用selenium来操作chrome来保持正常的资源加载。当然也可以选择用极其复杂的逻辑来解析得到资源网址，然后请求，但是我只有一上午的时间而且以前我也没学过爬虫。所以就选择了这个方法。
  2. 解析字符串
    在能够无障碍地得到各种资源后，下一步要做的就是得到我们需要的信息。这次我选择的是正则表达式，用正则表达式解析html文件得到各个信息。
